# Netbeans 12

## Instalación 

<p align="justify">

Para la instalación de este IDE utilizaremos los paquetes Snap usando el siguiente comando dentro de la terminal del ordenador (Fig.1):
  
</p>
 
</br>
</br>

**<p align="center"> sudo snap install netbeans --classic </p>**

</br>
</br>
