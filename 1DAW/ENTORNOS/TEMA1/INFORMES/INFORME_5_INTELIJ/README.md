# Itelij

## Instalación 

<p align="justify">

Para la instalación de este IDE se utilizarán los paquetes Snap. Esto se hará a través del comando (Fig.1): 
  
</p>
 
</br>
</br>

**<p align="center"> sudo snap install intellij-idea-community --classic </p>**

</br>
</br>
